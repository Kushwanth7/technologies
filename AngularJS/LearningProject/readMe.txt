This is a simple project implemented to learn the Angular Js technologies.
This project is implemented by looking at the Github tutorial -> https://github.com/angular/material-start/tree/es5-tutorial
This app is implemented to learn about the following
a) How Angular material and flex options can be used to configure html containers
b) Learning to use some of the Angular components such as <md-toolbar>,<md-sidenav>
c) Custom controllers such as <md-bottomsheet> with html templates
d) How custom controller can be used to programmatically open and close a sidenav component
e) Responsive breakpoints and $mdMedia are used
f) Theming can be altered/configured using $mdThemingProvider
